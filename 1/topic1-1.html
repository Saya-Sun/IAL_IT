📘 IGCSE IAL IT Topic 1
🔹 Features vs Functions

Feature:
• Physical parts and characteristics of a device
     Example:   camera, touchscreen

Function:
• What the device is used for (Purpose)
• the tasks or operations a device can perform
     Example:   calling, gaming

👉 Exam Tip:
Many questions start from this concept!

👉 Compare devices based on portability or performance

📘 Device Features & Functions

1 💼 Portability
Portability is how easy a device is to carry or move around.
  • Smaller and lighter devices are more portable
  • High → smartphones
  • Low → desktops
  • ✔ Easy to carry
  • ❌ Less performance
2 ⚡ Performance
Performance is how fast a device works.
  • CPU, RAM
  • SSD faster
  • ✔ Fast
  • ❌ Expensive
3 💾 Storage
Storage is data capacity.
  • HDD vs SSD
  • RAID 0 / RAID 1
  • ✔ More data
  • ❌ Cost
4 🖥️ User Interface
How user interacts with device.
  • GUI / CLI
  • ✔ Easy UI
  • ❌ Complex UI
5 🌐 Connectivity
How devices communicate.
  • USB, HDMI
  • Wi-Fi, Bluetooth
  • ✔ Flexible
  • ❌ Slower wireless
6 🎬 Media Support
Supports storage media.
  • USB
  • SD cards
  • CD/DVD
7 🔋 Energy
Power usage.
  • Battery life
  • Efficiency
8 🧩 Expansion
Ability to upgrade.
  • Add RAM
  • Add storage
9 🔒 Security
Protects data.
  • Passwords
  • Encryption
  • Biometrics
