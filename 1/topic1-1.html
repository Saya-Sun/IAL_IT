📘 IGCSE IAL IT Topic - 1

📘 Device Features & Functions

🔹 Features vs Functions

Feature:
• Physical or built-in part of device
• Example: camera, touchscreen

Function:
• What the device is used for
• Example: calling, gaming

👉 Exam Tip:
Many questions start from this concept!
🔹 Types of Devices

You MUST know:

  • Computers
  • Peripherals
  • Mobile devices
  • Storage devices
  • Embedded systems
👉 Compare devices based on portability or performance
💼 Portability
Definition:
Portability is how easy a device is to carry or move around.
  • Smaller and lighter devices are more portable
  • Important for mobile devices
  • High portability → smartphones, tablets
  • Low portability → desktop computers
  • ✔ Easy to carry anywhere
  • ❌ Smaller size may reduce performance
⚡ Performance
Performance is how fast and efficiently a device works.
  • CPU speed
  • RAM
  • SSD faster than HDD
  • ✔ Better experience
  • ❌ More expensive
💾 Storage
Storage is the ability of a device to store data.
  • HDD → cheaper, slower
  • SSD → faster, expensive
  • RAID 0 → speed
  • RAID 1 → backup
  • ✔ More storage
  • ❌ Higher cost
🖥️ User Interface
User interface is how a user interacts with a device.
  • GUI
  • CLI
  • ✔ Easy UI better
  • ❌ Complex = harder
🌐 Connectivity
Connectivity is how devices communicate.

Wired: USB, HDMI, VGA

Wireless: Wi-Fi, Bluetooth, NFC

  • ✔ Wired = stable
  • ✔ Wireless = flexible
  • ❌ Wireless slower
🎬 Media Support
Media support is ability to use storage media.
  • SD cards
  • USB drives
  • CD/DVD
🔋 Energy
Energy consumption is power usage.
  • Battery life
  • Environment
🧩 Expansion
Ability to upgrade device.
  • Add RAM
  • Add storage
🔒 Security
Protects data and access.
  • Passwords
  • Encryption
  • Biometrics
