📘 IGCSE IAL IT Topic 1
🔹 Features vs Functions
Feature: Physical parts of a device (camera, touchscreen)
Function: Tasks a device performs (calling, gaming)
👉 Common exam question – know the difference clearly!
🔹 Types of Devices
  • Computers
  • Peripherals
  • Mobile devices
  • Storage devices
  • Embedded systems
👉 Compare devices using portability & performance
📘 Device Features
1 💼 Portability
Ease of carrying a device.
  • Small & light = more portable
  • Phones > Laptops > Desktops
✔ Easy to carry | ❌ Less powerful
2 ⚡ Performance
How fast a device works.
  • CPU speed & cores
  • RAM size
  • SSD faster than HDD
✔ Fast tasks | ❌ Expensive
3 💾 Storage
Amount of data stored.
  • HDD vs SSD
  • RAID systems
✔ More data | ❌ Cost
4 🖥️ User Interface
How users interact with device.
  • GUI (easy)
  • CLI (advanced)
✔ Easy UI | ❌ Complex CLI
5 🌐 Connectivity
Communication between devices.
  • USB, HDMI
  • Wi-Fi, Bluetooth
✔ Flexible | ❌ Wireless slower
6 🎬 Media Support
Supported storage media.
  • USB, SD cards
  • CD/DVD
7 🔋 Energy
Power consumption.
  • Battery life
  • Efficiency
8 🧩 Expansion
Ability to upgrade device.
  • Add RAM
  • Add storage
9 🔒 Security
Protects data and access.
  • Passwords
  • Encryption
  • Biometrics
