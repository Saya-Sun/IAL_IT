📘 Device Features & Functions

🔹 FEATURES vs FUNCTIONS (INTRO)

Feature:
• Physical or built-in part of device
• Example: camera, touchscreen

Function:
• What the device is used for
• Example: calling, gaming

👉 Exam Tip:
Many questions start from this concept!

🔹 TYPES OF DEVICES (INTRO)

You MUST know:

  • Computers
  • Peripherals
  • Mobile devices
  • Storage devices
  • Embedded systems
👉 These are used in questions like:
“Compare devices based on portability or performance”

🔢 1. PORTABILITY

Definition:
Portability is how easy a device is to carry or move around.

Key Points

  • Smaller and lighter devices are more portable
  • Important for mobile devices

Examples

  • High portability → smartphones, tablets
  • Low portability → desktop computers

⚠️ Exam Evaluation

  • ✔ Easy to carry anywhere
  • ❌ Smaller size may reduce:
    • performance
    • screen size

🔢 2. PERFORMANCE

Definition:
Performance is how fast and efficiently a device works.

Key Factors

  • CPU speed
  • RAM
  • Storage type (SSD faster than HDD)

Example

  • Fast computer → opens apps quickly
  • Slow computer → delays and lag

⚠️ Exam Evaluation

  • ✔ High performance improves user experience
  • ❌ High performance devices are more expensive

🔢 3. STORAGE

Definition:
Storage is the ability of a device to store data and programs.

Types

  • HDD → large, cheaper, slower
  • SSD → fast, durable, expensive

Key Ideas

  • Capacity (GB/TB)
  • Speed
  • Reliability

🔥 RAID (Important)

  • RAID 0 → increases performance
  • RAID 1 → provides backup (mirroring)

⚠️ Exam Evaluation

  • ✔ More storage = store more data
  • ❌ Larger storage may increase cost and size

🔢 4. USER INTERFACE (UI)

Definition:
User interface is how a user interacts with a device.

Types

  • GUI → graphical (icons, windows)
  • CLI → text-based commands

Examples

  • Mouse → simple interface
  • Computer → complex interface

⚠️ Exam Evaluation

  • ✔ Simple UI = easy to use
  • ❌ Complex UI = harder but more powerful

🔢 5. CONNECTIVITY

Definition:
Connectivity is how devices connect and communicate with others.

Wired

  • USB
  • HDMI
  • VGA

Wireless

  • Wi-Fi
  • Bluetooth
  • NFC

💡 Key Exam Points

  • HDMI > VGA (better quality)
  • USB 3 > USB 2 (faster)
  • USB-C = modern universal standard

⚠️ Exam Evaluation

  • ✔ Wired → faster, stable
  • ✔ Wireless → flexible
  • ❌ Wireless may be slower

🔢 6. MEDIA SUPPORT

Definition:
Media support is the ability to use different storage media.

Examples

  • SD cards
  • USB drives
  • CDs/DVDs

💡 Modern Trend

  • Less optical disks
  • More cloud storage

⚠️ Exam Evaluation

  • ✔ More support = flexible
  • ❌ Less support = cheaper, thinner devices

🔢 7. ENERGY CONSUMPTION

Definition:
Energy consumption is how much power a device uses.

Importance

  • Battery life
  • Environmental impact

⚠️ Exam Evaluation

  • ✔ Low energy = longer battery life
  • ❌ High performance uses more energy

🔢 8. EXPANSION CAPABILITY

Definition:
Expansion capability is the ability to upgrade or add components.

Examples

  • Add RAM
  • Add storage (SD card)

⚠️ Exam Evaluation

  • ✔ Expandable devices last longer
  • ❌ Non-expandable devices are limited

🔢 9. SECURITY FEATURES

Definition:
Security features protect data and access.

Methods

  • Passwords
  • Encryption
  • Biometrics (fingerprint, face recognition)

💡 Key Point

  • Biometrics are harder to hack than passwords

⚠️ Exam Evaluation

  • ✔ Strong security protects data
  • ❌ May reduce convenience
