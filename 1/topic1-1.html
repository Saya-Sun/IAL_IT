📘 Device Features & Functions

🔹 FEATURES vs FUNCTIONS (INTRO)

Feature:
• Physical or built-in part of device
• Example: camera, touchscreen

Function:
• What the device is used for
• Example: calling, gaming

👉 Exam Tip:
Many questions start from this concept!
🔹 TYPES OF DEVICES (INTRO)

You MUST know:

  • Computers
  • Peripherals
  • Mobile devices
  • Storage devices
  • Embedded systems
👉 Compare devices based on portability or performance
🔢 1. PORTABILITY
Definition:
Portability is how easy a device is to carry or move around.

Key Points

  • Smaller and lighter devices are more portable
  • Important for mobile devices

Examples

  • High portability → smartphones, tablets
  • Low portability → desktop computers

⚠️ Exam Evaluation

  • ✔ Easy to carry anywhere
  • ❌ Smaller size may reduce performance and screen size
🔢 2. PERFORMANCE
Definition:
Performance is how fast and efficiently a device works.
  • CPU speed
  • RAM
  • SSD faster than HDD

⚠️ Exam Evaluation

  • ✔ Better experience
  • ❌ More expensive
🔢 3. STORAGE
Definition:
Storage is the ability of a device to store data and programs.
  • HDD → cheaper, slower
  • SSD → faster, expensive

🔥 RAID

  • RAID 0 → speed
  • RAID 1 → backup

⚠️ Exam Evaluation

  • ✔ More storage
  • ❌ Higher cost
🔢 4. USER INTERFACE
User interface is how a user interacts with a device.
  • GUI
  • CLI

⚠️ Exam Evaluation

  • ✔ Easy UI better
  • ❌ Complex = powerful but harder
🔢 5. CONNECTIVITY
Connectivity is how devices communicate.

Wired: USB, HDMI, VGA

Wireless: Wi-Fi, Bluetooth, NFC

⚠️ Exam Evaluation

  • ✔ Wired = stable
  • ✔ Wireless = flexible
  • ❌ Wireless slower
🔢 6. MEDIA SUPPORT
Media support is ability to use storage media.
  • SD cards
  • USB drives
  • CD/DVD
🔢 7. ENERGY
Energy consumption is power usage.
  • Battery life
  • Environment
🔢 8. EXPANSION
Ability to upgrade device.
  • Add RAM
  • Add storage
🔢 9. SECURITY
Protects data and access.
  • Passwords
  • Encryption
  • Biometrics
