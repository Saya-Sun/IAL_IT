Topic 1

🔹 Features Affect Design & Use

Device features affect size, performance, portability and usability.

💡 Exam Tip: Always link feature → effect
📱 Device Features
1

📍 GPS

Used for navigation and tracking
  • Uses satellites to find location
  • Small hardware → easy to include
  • ❗ Uses more battery
2

🔐 Biometrics

Improves security
  • Fingerprint / face recognition
  • Small size → easy to build into devices
3

📲 Touchscreen

Input + Output device
  • Saves space → smaller device
  • Larger display improves usability
4

🌡️ Sensors

Collect data automatically
  • Light sensor → brightness
  • Accelerometer → rotation
  • Motion sensor → steps
  • Temperature sensor
5

💾 Memory

Affects performance
  • RAM = temporary
  • ROM = permanent
  • More RAM → faster device
  • ❗ Larger size needed
6

📦 Storage

Stores data
  • More storage improves performance
  • ❗ Takes space in device
7

🔋 Battery

Affects usage time
  • Longer battery = better
  • ❗ Bigger battery reduces portability
8

🔬 Miniaturisation

Smaller but powerful devices
  • More transistors in IC
  • Devices become compact
9

⚙️ Processor

Controls performance
  • Higher clock speed = faster
  • More cores = multitasking
10

📡 RFID

Tracks items
  • Used to locate devices
11

📶 NFC

Short-range communication
  • File sharing
  • Contactless payment
12

🔳 QR Code

Stores data links
  • Scan to open web pages
13

🌐 Connectivity

Device connections
  • Wireless = portable
  • Wired = less portable
