🔹 What is Firmware?

Firmware is a type of software stored inside an embedded system that provides the basic instructions needed for the device to function.

Controls startup and basic input/output operations
💡 Exam Tip: Firmware = low-level control + stored on chip
⚙️ Functions of Firmware
1

🚀 Startup Control

Boot process

Initialises the system when powered on.

2

🔌 Hardware Control

Manages components

Controls communication between hardware parts.

3

📥 Input / Output

Basic operations

Handles simple input and output tasks.

💾 Where is Firmware Stored?

  • Embedded systems → stored on the chip
  • General-purpose computers → stored in ROM
Embedded systems keep firmware inside the device itself
🧠 Parts of Firmware
1

💻 BIOS

Basic Input/Output System
  • Starts up the computer
  • Initialises hardware
  • Prepares system for OS
2

📦 Bootloader

Loads operating system
  • Runs after BIOS
  • Loads the OS into memory

📌 Summary

Firmware is essential software that allows devices to start, control hardware, and perform basic operations before the operating system runs.
