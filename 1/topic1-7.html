💻 Processing Data

Devices process text, numbers, sound, and images.

All data must be converted into binary (0 and 1)
  • Humans → analogue
  • Computers → digital
💡 Always mention binary when explaining processing

🔢 Denary to Binary

Denary = base 10 | Binary = base 2

Binary values: 128, 64, 32, 16, 8, 4, 2, 1

  • Start from largest value
  • Fit → 1 | Not fit → 0

📏 Data File Size

File size depends on data type:

  • Text → small
  • Video/Image → large

Unit Equal to
