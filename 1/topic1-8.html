Topic 1

💻 Software

Software is all the instructions that tell a computer what to do.

Two main types: System Software & Application Software

📊 System vs Application Software

System Software Application Software
Runs and maintains the computer Performs tasks for the user
Controls hardware Used directly by users
Allows interaction with system Used to complete specific tasks
e.g: OS, drivers, antivirus e.g: Word, browser, games

⚙️ System Software

Allows the computer to run efficiently and control hardware.

  • Operating system
  • Antivirus
  • Firewall
  • Drivers
  • Disk defragmentation
  • Compression software

📱 Application Software

Allows users to perform tasks.

  • Word processor → documents, CV
  • Spreadsheet → budgets, calculations
  • Database → store & track data
  • Photo editor → image editing
  • Video editor → create videos
  • Web browser → internet access
  • Games → entertainment

🧠 Role of Operating System

Allows user interaction + manages system processes
  • Managing devices
  • Managing users
  • Managing processes
  • Managing security

🔌 Managing Devices

  • Controls peripherals (keyboard, mouse, printer, scanner, monitor)
  • Uses drivers to communicate with hardware
  • Handles interrupts from devices
💡 Driver = translates between device and computer

⚡ Managing Processes

  • Moves data between RAM and storage
  • Allocates memory to tasks
  • Prevents memory conflicts
  • Supports multitasking
Multitasking = running multiple tasks at once

👤 Managing Users

  • Provides GUI (windows, icons, menus, pointer)
  • Allows multiple user accounts
  • Manages files (open, delete, copy, move)
  • Runs application software

🔒 Managing Security

  • Usernames and passwords
  • Access levels and permissions
  • Prevents unauthorized access
  • Protects data from deletion
  • Uses antivirus, anti-malware, firewall
💡 Keeps data safe and confidential
