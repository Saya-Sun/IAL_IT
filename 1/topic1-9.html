📘 IGCSE IAL IT Topic 1

💻 Sources of Software

Software can be developed by individuals or organisations

📊 Types of Software Sources

Source Description
Free Users can study, modify and share code
(may not be free of cost)
Open-source Code is available, can be modified, usually free, community-developed
Proprietary Code is hidden, cannot modify,
developer provides support
Creative Commons Licences that control
how software can be used and shared
💡 Free ≠ Free of cost (means freedom)

🪪 Creative Commons Licences

  • Attribution → must credit creator
  • Share Alike → must use same licence
  • Non-commercial → no business use
  • No derivatives → cannot share modified version

📄 Software Licences

Licence Description
Single user One user, one computer
Multiple user Installed on limited number of computers
Institutional Used across organisation
Fixed term Used for limited time (subscription)
Indefinite One payment, use forever
Network Installed on server, shared across network

🔄 Updating Software

Software needs regular updates to stay effective.

  • Fix bugs
  • Add features
  • Improve security
  • Improve interface

⚙️ Types of Updates

Type Description
Patch Quick fix for bugs or security issues
Automatic Updates without user action
Upgrade New version with major improvements

⚠️ Update Issues

Compatibility problems may occur
  • Software may not work on older hardware
  • New OS may not support old apps
