📘 IGCSE IT Topic 2 - Networks

🌐 What is a Network?

A network is simply two or more devices that have been connected to each other.

💡 Exam Tip: Always define clearly + simple
🔗 Network Components
1

💻 Network Interface Card (NIC)

Connects device to network
  • Built-in circuit board
  • Wired → Ethernet
  • Wireless version = WNIC
2

🔌 Hub

Sends data to ALL devices
  • Creates heavy traffic
  • Causes collisions
  • Reduces performance ❗
3

🔀 Switch

Sends data to correct device only
  • Uses MAC address
  • Stores device table
  • Reduces collisions → better performance
4

📡 Router

Finds best path for data
  • Uses IP address
  • Uses routing table
  • Chooses best route
5

🌍 Modem

Converts signals
  • Analogue ↔ Digital
  • Used in internet connection
6

🌐 Gateway

Translates between networks
  • Different protocols
  • Acts like translator
7

🌉 Bridge

Connects LANs together
  • Creates larger LAN
  • Uses MAC address
8

📶 Repeater

Boosts signal strength
  • Extends distance
  • Fixes weak signals
9

🖥️ Server

Provides data/services
  • Stores web pages
  • Handles requests
  • Can manage files, printing
10

📡 Wireless Access Point (WAP)

Provides wireless connection
  • Uses radio signals
  • Works with WNIC

⚠️ Important Note

Many devices (like home routers) combine multiple components such as modem, router, gateway, switch and WAP.

💡 Exam Tip: Know differences → Hub vs Switch is VERY common!
