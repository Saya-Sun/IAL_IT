🌐 Types of Network Models

A network model describes how devices are structured and how they communicate.

💡 Exam Tip: Always include structure + control + example
📊 Network Models
1

🖥️ Client-Server Network

Central server controls all clients

Structure: Clients send requests → Server responds

Examples:

  • File server
  • Web server
  • Print server

Advantages:

  • Centralised security
  • Automatic backups
  • Easy updates
  • Reliable performance

Disadvantages:

  • Expensive ❗
  • Needs specialist staff
  • If server fails → network stops
2

🔗 Peer-to-Peer Network

No central server • Equal devices

Structure: Devices share directly

Advantages:

  • Cheap
  • Easy to set up
  • No specialist needed

Disadvantages:

  • Less secure
  • No central backup
  • Slow with many devices
3

📡 Ad Hoc Network

Temporary • No infrastructure

Use: Short-term connections

Advantages:

  • Quick setup
  • No equipment needed

Disadvantages:

  • Weak security
  • Limited size
4

📱 Tethering

Phone acts as hotspot

Connections: Wi-Fi, Bluetooth, USB

Advantages:

  • Internet anywhere
  • More secure than public Wi-Fi

Disadvantages:

  • Drains battery
  • Uses mobile data

📊 Comparison Table

Feature Client-Server Peer-to-Peer
Control Central None
Security High Low
Cost High Low
Performance Stable Slower with many devices
