📡 Communication Protocols

A protocol is a set of rules that govern how data is transmitted in a network.

💡 Exam Tip: Always define protocol clearly → "rules for communication"
📶 Types of Protocols
1

🔌 Ethernet

Wired communication in LAN
  • Uses cables (twisted pair, coaxial, fibre optic)
  • Invented by Robert Metcalfe (1973)
  • Speed up to 100 Gbps
  • Very reliable (error detection)
  • More secure than wireless
  • Used in offices, schools
2

📡 Wi-Fi

Wireless communication using radio waves
  • Uses 2.4 GHz (long range) & 5 GHz (faster)
  • Range: ~45m (2.4GHz), ~15m (5GHz)
  • Speed up to 10 Gbps
  • Can suffer interference
  • Less secure than Ethernet
  • Used in homes, public networks
3

🔵 Bluetooth

Short-range wireless communication
  • Uses 2.4 GHz frequency
  • Range up to 10m
  • Speed up to 3 Mbps
  • Requires pairing (PIN sometimes)
  • Connects up to 7 devices
  • Used for headphones, keyboards
4

💡 ZigBee®

Low power IoT communication
  • Uses 2.4 GHz radio waves
  • Range up to 100m
  • Speed up to 250 Kbps
  • Very low power usage
  • Requires ZigBee hub
  • Used in smart homes
5

📱 Cellular

Wide-area mobile communication
  • Uses cells and base stations
  • Uses radio waves + microwaves
  • Data moves between base stations
  • 2G → 5G evolution
  • 5G speeds up to 10 Gbps
  • Used for calls, mobile internet
6

🔴 Infrared

Line-of-sight communication
  • Uses infrared light
  • Requires direct line of sight
  • Very short range
  • Cannot pass through walls
  • Weak signal
  • Used in remote controls

⚖️ Protocol Comparison

Ethernet = fastest & secure
Wi-Fi = flexible wireless
Bluetooth = short-range devices
ZigBee = IoT & low power
Cellular = large-area coverage
