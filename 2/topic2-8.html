📘 IGCSE IAL IT Topic 2

🔒 Network Security

Devices connected to a network are vulnerable to attacks and data theft.

💡 Exam Tip: Always mention hacker + malware threats
⚠️ Security Threats

⚠️ Threats

Risks when connected to a network
  • Hacking (unauthorised access)
  • Malware (virus, spyware, ransomware)
  • Fraud & identity theft
  • Data corruption or deletion
  • Open networks increase risk ❗
🛡️ Security Solutions
1

🔑 Authentication

Verifies user identity
  • Password → simple login
  • Two-factor → extra code for security
  • Biometrics → fingerprint, face, iris
2

⚙️ Security Settings

Control access and privacy
  • Make data private
  • Enable two-factor authentication
  • Limit who can view data
3

👤 Access Rights

Limits what users can do
  • Read-only or edit access
  • Prevents accidental damage
  • Reduces impact of hacking
4

🦠 Anti-Malware

Detects and removes malware
  • Scans files and system
  • Deletes or quarantines threats
  • Must be regularly updated ❗
5

🔥 Firewall

Controls incoming/outgoing traffic
  • Uses rules (criteria)
  • Blocks suspicious data
  • Hardware = safer
  • Software = easier to install
6

🔐 Physical Security

Protects hardware
  • Locks, alarms
  • Keycards
  • CCTV
7

🔑 Encryption

Scrambles data
  • Makes data unreadable
  • Uses encryption key
  • Symmetric → same key
  • Asymmetric → different keys

⚖️ Key Idea

Authentication = who are you
Firewall = block threats
Encryption = protect data
💡 Combine multiple security methods for best protection
