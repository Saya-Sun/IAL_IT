🌐 Network Components

What is a Network?

A network is two or more devices connected together to share data and resources.

💡 Exam Tip: Always include "share data/resources" in your definition.
1

💻 Network Interface Card (NIC)

Connects a device to a network
  • Built-in hardware
  • Wired (Ethernet) or Wireless (WNIC)
2

🔌 Hub

Broadcasts data to all devices
  • High traffic
  • Many collisions
  • Poor performance
3

🔀 Switch

Sends data to the correct device only
  • Uses MAC address
  • Learns MAC address table
  • Fewer collisions → faster
4

📡 Router

Chooses best path for data
  • Uses IP address
  • Routing table
5

🌍 Modem

Converts analogue ↔ digital signals
6

🌐 Gateway

Translates between different networks
7

🌉 Bridge

Connects LANs together
8

📶 Repeater

Boosts signal strength
9

🖥️ Server

Provides data and services
10

📡 Wireless Access Point (WAP)

Provides wireless network access

📊 Simple Network Flow

Devices → Switch → Router → Internet

⚖️ Hub vs Switch

Feature Hub Switch
Transmission All devices Specific device
Address Used None MAC address
Traffic High Low
Collisions Many Few
Performance Slow Fast
💡 Exam Tip: Hub = broadcast to ALL, Switch = send to ONE device using MAC address.

🧠 Quick Memory Summary

  • NIC → Connects device
  • Hub → Sends to ALL
  • Switch → Sends to ONE
  • Router → Finds best path
  • Modem → Converts signals
  • WAP → Wireless access

⚠️ Important Note

Modern home routers often combine multiple devices such as router, switch, modem, gateway, and WAP.

💡 Exam Tip: Questions often test device differences (especially Hub vs Switch).
