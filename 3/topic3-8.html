☁️ The Cloud

📦 Cloud Storage

Data stored on remote servers

Cloud storage is a method of storing data on remote servers that are accessed via the internet, instead of storing data locally on a device.

Key Idea: Data is stored by a third-party provider

💡 Exam Tip: Cloud storage = remote storage via internet

💻 Cloud Computing

Storage + services (software & resources)

Cloud computing includes cloud storage along with additional services such as software, processing power and backups.

📱 Examples

  • Microsoft 365 – Word, Excel online
  • iCloud – stores photos, files, contacts
✅ Advantages
1

💰 Cost Saving

No need for hardware

Users do not need to buy expensive storage devices.

2

👨‍💻 Less Maintenance

No technical staff needed

Cloud provider manages hardware and systems.

3

🌍 Accessibility

Access anywhere

Data can be accessed from any device with internet.

4

💾 Backup

Automatic backups

Data is regularly backed up by the provider.

5

🔄 Updates

Automatic software updates

Users always have the latest version.

⚠️ Disadvantages
1

💳 Cost

Subscription fees

Large storage or services require payment.

2

🔓 Security Risks

Trust provider

Data may be hacked or misused if security is weak.

3

🌐 Internet Dependence

Requires connection

Cannot access data without internet or with poor connection.

🧠 Quick Memory

  • Cloud storage → remote data
  • Cloud computing → storage + services
  • + Access anywhere
  • – Needs internet
