🖥️ Main Components of an IT System

📌 What is an IT System?

An IT system is a combination of components that work together to perform tasks.

1

Hardware

Physical devices
  • Computers
  • Scanners
  • Printers
  • Input/output devices
2

Software

Programs and applications
  • Operating system
  • Database software
  • User interface
  • Utility software
3

Processes

Steps and actions
  • Input data
  • Process data
  • Store and retrieve data
  • Output results
4

People

Users of the system
  • Customers
  • Employees
  • Administrators
💡 Exam Tip: IT system = Hardware + Software + Processes + People

✈️ Example: Airport Check-in System

  • Hardware: scanner, scale, printer
  • Software: database, OS, interface
  • Processes: scan ticket, store data
  • People: passengers and staff
