📊 Data and Information
1

Data

Raw facts without meaning

Data can be numbers, text, images, sounds, or symbols that have no context.

Example: 1008

2

Information

Data with meaning

Information is data that has been processed and given context so it makes sense.

Example: There are 1008 official Pokémon.

💡 Exam Tip: Data + Context = Information

⚖️ Data vs Information

  • Data: raw, unprocessed
  • Information: processed and meaningful

🔄 Data Processing

Turning data into information
  • Collect data
  • Process data
  • Produce information

🧠 Summary

Data is raw and meaningless, while information is processed data that has meaning and can be used for decision-making.

💡 Exam Tip: Information helps decision-making
