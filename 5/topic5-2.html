📂 Structured vs Unstructured Data
1

Structured Data

Organised with a fixed format
  • Stored in databases or tables
  • Has a clear structure (e.g. rows and columns)
  • Easy to search, sort, and analyse
📌 Examples
  • Product price: $1.95
  • Email date and time
  • Spreadsheet data
🤖 Use in AI

Machine learning uses structured data (e.g. plant colour, height, type) to learn patterns.

2

Unstructured Data

No fixed format
  • Mixed types of data
  • Harder to search and analyse
  • Often qualitative
📌 Examples
  • Email message content
  • Social media posts
  • Images, videos

⚖️ Comparison

Feature Structured Unstructured
Format Fixed No fixed format
Search Easy Difficult
Storage Databases Files, media
Type Quantitative Often qualitative

🏫 School Management System

i

Structured Data

Organised and easy to search
  • Student ID (e.g. 2025-001)
  • Student name
  • Class (e.g. Year 10)
  • Exam marks (e.g. 85%)
  • Attendance records

These are stored in databases and can be easily searched, sorted, and analysed.

ii

Unstructured Data

No fixed format
  • Teacher comments about students
  • Emails between teachers and parents
  • Student essays or projects
  • Photos or videos of school events

These are harder to search and analyse because they do not follow a fixed structure.

💡 Exam Tip:
Database = structured,
text/media = unstructured

🧠 Summary

Structured data is organised and easy to analyse, while unstructured data is flexible but harder to process.
