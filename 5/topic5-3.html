📌 What is Data Extraction?

Data extraction is the process of selecting useful data from large datasets and processing it to produce meaningful information.

Process

Turning data into information
  • Collect large amounts of data
  • Select useful data
  • Process and analyse it
  • Produce meaningful information
💡 Extract → Process → Decide
2

⛏️ Data Mining

Finding patterns in large datasets

Data mining is the process of analysing large amounts of data to discover patterns, trends, and useful information.

How Data Mining Works

  • Collect large amounts of data
  • Use software to analyse data
  • Identify patterns and relationships
  • Use results for decision-making

🛒 Shopping

Stores analyse buying patterns to recommend products.

📱 Social Media

Platforms analyse user activity to suggest content.

🏥 Healthcare

Hospitals analyse patient data to improve treatment.

💡 Data mining = finding patterns, not just collecting data

⚖️ Data Mining vs Data Extraction

  • Data Mining: finds patterns
  • Data Extraction: selects useful data
💡 Mining = patterns, Extraction = selecting data

📊 Scenario : Clothing Company

A company collects customer data about favourite colours.

  • Remove unwanted data (black, cream)
  • Analyse remaining responses
  • Select top 3 most popular colours

This helps decide which products will sell best.

1

📈 Better Decisions

Helps organisations make informed choices.

🛍️ Product Improvement

💰 Increased Sales

Focus on popular trends to boost revenue.

4

🚀 Business Growth

🏢 Other Uses

  • Marketing strategies
  • Service improvements
  • Competitor analysis
  • Predicting trends
💡 Data extraction helps businesses stay competitive

🧠 Summary

Data extraction converts large amounts of raw data into useful information that organisations use for decision-making and improvement.

Data mining is used to analyse large datasets and discover patterns that help organisations improve decisions and performance.
