💻 SQL (Structured Query Language)

📌 What is SQL?

Used to query databases

SQL is a language used to search, retrieve and manage data in a database.

🔍 SELECT & FROM

SELECT SongTitle FROM Songs; SELECT SongTitle, Genre FROM Songs; SELECT * FROM Songs;
💡 * means select all fields

🎯 WHERE

SELECT SongTitle FROM Songs WHERE Genre = 'Pop'; SELECT SongTitle FROM Songs WHERE Sales > 10000;

Used to filter results.

⚙️ AND / OR / NOT

SELECT SongTitle FROM Songs WHERE Artist='T. Singer' AND Genre='Pop'; SELECT SongTitle FROM Songs WHERE Genre='Pop' OR Genre='Rock'; SELECT SongTitle FROM Songs WHERE NOT Genre='Rock';

➕ INSERT

INSERT INTO Songs VALUES (005, 'Forever', 'The 2 Rs', 'Rock', 5000);

✏️ UPDATE

UPDATE Songs SET Artist='The 3 Rs' WHERE Artist='The 2 Rs';

❌ DELETE

DELETE FROM Songs WHERE SongID=005;

🔗 INNER JOIN

SELECT Songs.SongID, Artists.Artist FROM Songs INNER JOIN Artists ON Songs.ArtistID = Artists.ArtistID;

Used to combine data from multiple tables.

📊 COUNT & GROUP BY

SELECT COUNT(*) FROM Songs; SELECT COUNT(SongID), Genre FROM Songs GROUP BY Genre;

📈 ORDER BY

SELECT * FROM Songs ORDER BY Genre ASC; SELECT * FROM Songs ORDER BY Sales DESC;

🔎 Wildcards

SELECT * FROM Artists WHERE Artist LIKE 'T%'; SELECT * FROM Artists WHERE Artist LIKE '%es'; SELECT * FROM Songs WHERE Genre LIKE 'P__';
💡 % = many characters, _ = one character

🧠 Summary

  • SELECT → retrieve data
  • WHERE → filter data
  • JOIN → combine tables
  • INSERT/UPDATE/DELETE → modify data
