Data Integrity

Quick Study Guide

Definition

Data integrity refers to how reliable and trustworthy data is. It ensures data remains accurate, consistent, and error-free throughout its lifecycle.

Importance

  • Supports effective business decisions
  • Ensures data is reliable and trustworthy
  • Maintained through data governance

Key Components

Accuracy

Data must be correct and free from errors.

Consistency

Data should follow the same format and rules.

Completeness

All required data must be present with no missing values.

Reliability

Data must be available and trusted when needed.

Security

Protects data from unauthorized access or changes.

Maintaining Data Integrity

Data Validation

Checks data for errors before entry.

Data Backup

Creates copies of data to prevent loss.

Access Controls

Limits who can view or edit data.

Audit Trails

Tracks who changed data, when, and what was changed.

Summary

  • Data integrity = accuracy + consistency + completeness + reliability + security
  • Maintained using validation, backup, access control, and audit trails
