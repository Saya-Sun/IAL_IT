Topic 1: Hardware & Software

1. Features and Functions of Modern Devices

  • Computers: Process data, run software, perform calculations
  • Peripherals: Input/output devices like keyboard, mouse, printer
  • Mobile Phones: Communication, apps, internet access
  • Storage Devices: HDD, SSD, USB – store data permanently
  • Embedded Systems: Built into devices like washing machines, cars
